Some day
========
A game about cats, little blue boys and astronauts.

Compile and run
---------------
Download and install [node.js](https://nodejs.org/).

Open terminal and type

    $ cd path/to/this/folder/
    $ npm install
    $ npm start

Open your browser to http://localhost:8000/
